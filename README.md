# 节气公转可视化 WebApp

这是一个用于展示“地球绕太阳公转”的顶视图 WebApp 项目，重点表现椭圆轨道、角度刻度与 24 节气在轨道上的位置关系。点击节气标记可弹出节气介绍卡片。

## 目标
- 用清晰美观的方式解释地球公转的椭圆轨道。
- 标注 0°-360° 角度刻度与 24 节气位置。
- 点击节气即可查看简介卡片。

## 需求文档
详见 `docs/requirements.md`。

## 当前状态
- 已完成需求文档与首版页面
- 基于 SVG 绘制椭圆轨道、角度刻度与 24 节气标记
- 点击节气可弹出介绍卡片

## 核心功能
- 椭圆轨道与太阳焦点可视化
- 角度刻度（每 15°）与 0°-360°标注
- 24 节气标记与交互卡片
- 轻量动效（地球示意运动）

## 本地运行
直接打开 `index.html` 即可（无需依赖）。也可以使用任意静态服务器。

示例：
```bash
python3 -m http.server 8080
```
