# 节气公转可视化 WebApp

这是一个用于展示“地球绕太阳公转”的顶视图 WebApp 项目，重点表现椭圆轨道、角度刻度与 24 节气在轨道上的位置关系。点击节气标记可弹出节气介绍卡片。

## 目标
- 用清晰美观的方式解释地球公转的椭圆轨道。
- 标注 0°-360° 角度刻度与 24 节气位置。
- 点击节气即可查看简介卡片。
